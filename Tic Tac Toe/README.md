# `Tic-Tac-Toe Game`

## Game developed with `JavaScript, HTML, CSS`
### *Rules of the Game*

- 2 player game

- 3 x 3 grid is provided to you 

- Player 1 symbol = O

- Player 2 symbol = X 

- You have to enter your symbol in such a way that there are 3 same symbols horizontally/ vertically/ diagonally
